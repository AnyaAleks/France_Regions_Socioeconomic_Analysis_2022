# Анализ социально-экономических показателей регионов Франции (2022)

Проект представляет собой комплексное статистическое исследование социально-экономического неравенства регионов Франции на основе официальных данных INSEE за 2022 год. Включает сбор, обработку, визуализацию данных и применение непараметрических статистических критериев для проверки гипотез.

## Цель проекта
Провести пространственный анализ регионов Франции, выявить статистически значимые различия и взаимосвязи между ключевыми показателями:
- ВВП на душу населения
- Уровень безработицы
- Средняя заработная плата
- Доля населения с высшим образованием
- Плотность населения

## Методология

### Используемые данные
1. **DS_RP_LOGEMENT_PRINC** – характеристики жилищного фонда (среднее число комнат)
2. **DS_RP_DIPLOMES_PRINC** – уровень образования (доля населения 25–34 лет с высшим образованием)
3. **DS_RP_ACTIVITE_PRINC** – занятость населения (доля работающих неполный день)
4. **DD_EEC_ANNUEL** – уровень безработицы по методологии МОТ
5. **Дополнительные данные** – ВВП на душу населения, плотность населения

### Методы анализа
1. **Предобработка данных**: обработка пропусков, стандартизация, нормализация
2. **Визуализация**: гистограммы, тепловые карты корреляций, диаграммы рассеяния
3. **Статистические критерии**:
   - Тест Шапиро-Уилка (проверка нормальности распределения)
   - Корреляционный анализ Спирмена
   - U-критерий Манна-Уитни (сравнение двух групп)
   - H-критерий Краскела-Уоллиса (сравнение трёх и более групп)

## Ключевые выводы
1. **Распределение ВВП на душу населения и средней зарплаты** не является нормальным из-за сильного влияния региона Île-de-France (Париж и окрестности)
2. **Обнаружена сильная положительная корреляция** между ВВП на душу населения и средней зарплатой (ρ = 0.988)
3. **Выявлена сильная отрицательная корреляция** между уровнем безработицы и средней зарплатой (ρ = -0.868)
4. **Нет статистически значимых различий** в ВВП на душу населения между группами регионов с высокой и низкой безработицей (p = 0.0528)

## Установка и запуск

### Предварительные требования
- Python 3.9 или выше
- pip (менеджер пакетов Python)
- Git (для клонирования репозитория)

### Шаг 1: Клонирование репозитория
```bash
git clone https://github.com/AnyaAleks/France_Regions_Socioeconomic_Analysis_2022.git
cd France_Regions_Socioeconomic_Analysis_2022
Шаг 2: Установка зависимостей
bash
pip install -r requirements.txt
Шаг 3: Получение данных
Важно: Исходные данные INSEE не включены в репозиторий из-за ограничений размера файлов на GitHub

Скачайте исходные данные с официального сайта INSEE:

DS_RP_LOGEMENT_PRINC_2022_data.csv

DS_RP_DIPLOMES_PRINC_2022_data.csv

DS_RP_ACTIVITE_PRINC_2022_data.csv

DD_EEC_ANNUEL_2024_data.csv

Соответствующие метаданные (.metadata.csv)

Поместите скачанные файлы в папку data/raw/

Шаг 4: Запуск анализа через командную строку
bash
python src/main.py
```
